# hyeongseok's Blog
hexo file



### 설치 방법

```bash
$ git clone git@github.com:hsoh1990/git-blog.git hsoh1990.github.com
$ cd hsoh1990.github.com 
$ npm install

$ git clone --recursive git@github.com:hsoh1990/tranquilpeak.git
$ cd themes/tranquilpeak
$ npm install -g bower grunt-cli
$ npm install
$ bower install
$ grunt build
```



### 로컬 실행 

```bash
$ hexo clean
$ hexo generate
$ hexo server
```



### 배포

```bash
$ hexo deploy
```



### 새글 작성

```bash
$ hexo new post <title>
```
